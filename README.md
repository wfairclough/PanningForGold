PanningForGold
==============

A small game written in Racket (Scheme) to practice functional programming.


How to use
----------

1. Download [Racket](http://racket-lang.org/download/)
2. Open panning-for-gold.rkt with the DrRacket IDE
3. Hit Run and have fun!
